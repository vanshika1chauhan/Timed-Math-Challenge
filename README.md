# ⏱️ Timed Math Challenge (Python)

## 📌 Description

The **Timed Math Challenge** is a beginner-friendly Python project that tests the user's speed and accuracy in solving random multiplication problems within a limited time.

The program generates random multiplication questions, records the time taken to answer each question, and displays the final performance at the end. This project helped me practice Python fundamentals such as loops, functions, random number generation, user input, conditional statements, and time handling.

## ✨ Features

* Generates random multiplication questions.
* Measures the time taken to complete the challenge.
* Displays the user's final score and total time.
* Beginner-friendly Python project.

## 🛠️ Technologies Used

* Python 3
* random module
* time module

## ▶️ How to Run

1. Download or clone this repository.
2. Open the project folder in VS Code or any Python IDE.
3. Run the program using:

   ```bash
   python timed_math_challenge.py
   ```
4. Answer the questions as quickly and accurately as possible.
